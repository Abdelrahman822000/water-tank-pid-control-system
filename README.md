# Water Tank PID Control System

A PID-based control system for regulating water level and temperature using MATLAB, Simulink, and Arduino.

## 📌 Project Overview

This project implements a real-time control system using PID (Proportional–Integral–Derivative) controllers to maintain stable water level and temperature.

The system was designed and simulated in MATLAB/Simulink, then deployed on Arduino for real-world testing.

## ⚙️ Features

- Real-time PID control for water level
- Temperature control system
- MATLAB/Simulink modeling and simulation
- Arduino-based hardware implementation
- System response analysis (overshoot, settling time, stability)

## 🧠 What I Did

- Designed PID controllers for level and temperature systems
- Built and simulated system using Simulink
- Tuned PID parameters for optimal performance
- Integrated Arduino for real-time control
- Analyzed system response and stability

## 🛠️ Technologies

- MATLAB
- Simulink
- Arduino
- Control Systems (PID)

## 📊 System Concept

The system uses feedback control:
1. Sensors measure current water level and temperature  
2. Controller compares with desired setpoint  
3. PID adjusts system input to minimize error
